# HSD – Chip Design (CHD)

Dieses Repository enthält alle Übungen und Materialien für das Fach **Chip Design (CHD)** im Rahmen meines Studiums.

## Projektstruktur

```
UebungX/
├── src/          # VHDL-Quelldateien
├── sim/          # Simulationsumgebung (ModelSim)
├── doc/          # Dokumentation und Unterlagen
└── literature/   # Fachliteratur und Skripte
```

## Technologien

- **VHDL** – Hardwarebeschreibungssprache für alle Übungen
- **ModelSim ASE** – Simulation und Verifikation der VHDL-Designs

## Simulation starten

ModelSim kann über das Skript `sim/modelsim.sh` gestartet werden:

```bash
cd sim
./modelsim.sh
```
